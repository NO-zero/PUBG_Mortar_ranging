<h5>如果未输入一格的距离，则默认是100，请在一格的距离值为100的时候，将游戏地图缩放到合适的大小，然后测量地图上最小的一格的距离，会显示在”距离是：“后面
，这个一格的距离在游戏中就代表着100m</h5>
